### 介绍：
	使用Python写的一个简单的区块链

### 使用：

创建创世区块

```python
python cli.py create_genesis
```

挖矿

```python
python cli.py mine
```

显示区块链上的所有区块

```python
python cli.py show_blockchain
```

从其他节点同步区块链

```python
python cli.py
```
